Herramienta CAT (COCOMO AID TOOL)

La herramienta

Esta herramienta permite hacer las estimaciones de esfuerzo y duración de un proyeco de software, aplicando el modelo de COCOMO II. Fue desarollada para la materia Administración de Proyectos dentro de la Mastría de Ingeniería de Software de la UNLP. Actualmente se encuentra en desarrollo una versión mejorada de esta herramienta.

COCOMO II (Constuctive Cost Model)

Es un modelo empírico de estimación basado en la experiencia con proyectos grandes. Es un modelo de tres niveles que permite estimaciones cada vez mas detalladas y que pueden realizarse a la vez que progresa el desarrollo del proyecto.

Nivel de estimación

  • Nivel inicial de prototipado:

    • Las estimaciones de tamaño se basan en puntos de objeto y se utiliza una fórmula sencilla tamaño / productividad para estimar el esfuerzo requerido.
    Para realizar el cálculo con este nivel haga click aqui Cálculo con nivel inicial de prototipado
  • Nivel inicial de diseño:

    • Este nivel corresponde a la terminación de los requerimientos del sistema con (quizás) algún diseño inicial. Las estimaciones se basan en puntos de función que se convierten en un número de líneas de código fuente. La fórmula contiene un conjunto sencillo de multiplicadores asociados a ella.
    Para realizar el cálculo con este nivel haga click aqui Cálculo con nivel de diseño inicial
  • Nivel post-arquitectónico:

    • Una vez diseñada la arquitectura del sistema se hace una estimación razonablemente precisa del tamaño del software. La estimación en este nivel utiliza un conjunto mas grande de multiplicadores que reflejan la capacidad del personal, el producto y las características del proyecto.
    Para realizar el cálculo con este nivel haga click aqui Cálculo con nivel post-arquitectónico
